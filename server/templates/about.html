About Us

A brief explanation of our project

Our project is a research project in the digital humanities field, and it took place under the guidance of Dr. Yael Netzer in the Digital Humanities Course in the BGU University of The Negev.

This project was intended to try something that has never been done before: applying Sentiment Analysis (a language processing technique) to a work of arts, specifically a movie, therefore combining the digital world with the world of the humanities, just like the course intended. Our project is a creative project, and it's goal is to be a cool tool for people to use, but also to show the how accurate (or not sometimes) textual analysis can be, and how important (or not sometimes) is non verbal communication in our society.

We developed this project using Flask - a python web development library, with the help of Bootstrap, Jquery and the Highcharts library to enhance the UI and user experience. Searching the movies was done through the TMDB movie API, and we downloaded the subtitles from the OpenSubtitles API. The sentiment analysis of the subtitles was done with TextBlob, and scoring algorithm and normalization was achieved with the help of a small PostgreSQL database and Welford's algorithm for calculating a running mean and standard deviation (further credits are in the source code).

We are Guy Shuster, Ori Benami and Tomer Elgavish, and we hope you enjoy!
